# Singly Linked List Implementation in Python

## 📌 Overview

This project implements a **Singly Linked List** in Python as part of a Data Structures and Algorithms assignment. The implementation demonstrates core linked list operations, including insertion, deletion, traversal, and searching, along with additional advanced functionalities.

The project is designed to showcase understanding of **dynamic data structures**, pointer manipulation, and algorithm efficiency.

---

## ⚙️ Features

### ✅ Core Operations

* **append(data)** – Add a node at the end of the list
* **prepend(data)** – Insert a node at the beginning
* **insert_at_position(data, pos)** – Insert a node at a specific index
* **delete_by_value(value)** – Remove the first occurrence of a value
* **search(value)** – Find the index of a value
* **display()** – Print all elements in the list
* **is_empty()** – Check if the list is empty
* **size()** – Count the number of nodes

---

### 🚀 Advanced Features (Bonus)

* **reverse()** – Reverse the linked list
* **find_middle()** – Find the middle element using the two-pointer technique
* **detect_cycle()** – Detect cycles using Floyd’s Cycle Detection Algorithm

## ⏱️ Time Complexity Summary

| Operation          | Complexity |
| ------------------ | ---------- |
| Prepend            | O(1)       |
| Append             | O(n)       |
| Insert at Position | O(n)       |
| Delete by Value    | O(n)       |
| Search             | O(n)       |
| Reverse            | O(n)       |
| Find Middle        | O(n)       |
| Detect Cycle       | O(n)       |
